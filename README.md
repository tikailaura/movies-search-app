# Movies Search App
Delivered a dynamic frontend search application that connects user input to live API data, transforming JSON responses into a responsive movie results interface. Implemented asynchronous data fetching, dynamic DOM rendering, and user feedback states for loading, errors, and empty results — demonstrating practical experience with real-world API-driven UI behavior.

## Features

- Search for movies by title
- Fetch live data from the OMDb API
- Display movie posters, titles, year, and type
- Show loading state while searching
- Show error messages when requests fail
- Show a when no movies are found
- Support search by button click or Enter key
- Render results dynamically as movie cards

## Tech Stack

- HTML5
- CSS3
- JavaScript
- Fetch API
- OMDb API

## API Key Setup

This project uses the OMDb API to fetch movie data.

To run the project, you'll need your own free API key.

### Steps

1. Visit https://www.omdbapi.com/apikey.aspx
2. Request a free API key using your email address.
3. Once you receive your key, open the JavaScript file.
4. Replace the placeholder with your API key:

```javascript
const API_KEY = "YOUR_API_KEY";
```

5. Save the file and run the project.

> **Note:** The API key is not included in this repository for security reasons.
